# Aba Life - Monorepo

## Estrutura do Projeto

- `backend/`: API Node.js com Sequelize (versão robusta)
- `apps/passageiro/`: Aplicativo do passageiro
- `apps/motorista/`: Aplicativo do motorista
- `apps/admin/`: Painel administrativo web

## Comandos úteis

```bash
cd backend
npm install
npm start
```
